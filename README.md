# Modern C/C++ Project Structure

## Software Stack
- C/C++ Compiler: gcc/clang
- C/C++ Linker: mold/gld/gold/lld
- Library Interface: pkgconf/pkg-config
- Build System: muon/meson + samurai/ninja
- Documentation: doxygen
- Tests: Non opionated, left to the user
- Dependency Management: Nix flakes

## Project Structure Overview
```
.
├── src/                    # Source files
├── include/                # Header files
├── tests/                  # Unit/Integration/... tests
├── docs/                   # Documentation generated by doxygen
├── meson.build             # Top-level Meson build file
├── Doxyfile                # Configuration file for doxygen
├── Makefile                # Settings and development commands
├── flake.nix               # Environment and dependency management
├── .gitignore              # Git ignore file
└── README.md               # Project readme
```
