# Modern C/C++ Project Structure

## Stack
- C/C++ Compiler: gcc/clang
- C/C++ Linker: mold (optional)
- Library Interface: pkgconf/pkg-config
- Build System: muon/meson + samurai/ninja
- Auto Documentation: doxygen
- Package Manager: conan2
